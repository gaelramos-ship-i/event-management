# Projet Gestion d'Événements

## Description

Ce projet est une application web dédiée à l'exercice des bases de données avec **PHP** et **MySQL**. Il permet de pratiquer les concepts fondamentaux de la gestion de bases de données à travers une gestion simple de profils utilisateurs et d'authentification.

## Objectifs Pédagogiques

- Maîtriser la connexion aux bases de données MySQL avec PHP
- Pratiquer les opérations CRUD (Create, Read, Update, Delete)
- Mettre en place un système d'authentification
- Comprendre la gestion des sessions utilisateur
- Appliquer les bonnes pratiques en matière de sécurité (hachage des mots de passe, prévention des injections SQL)

## Structure du Projet

```
events/
├── index.php              # Page d'accueil
├── config/
│   └── connect.php        # Configuration et connexion à la base de données
├── users/
│   ├── login.php          # Page de connexion
│   ├── logout.php         # Déconnexion
│   ├── profile.php        # Profil utilisateur
│   └── register.php       # Inscription
└── README.md              # Ce fichier
```

## Technologies Utilisées

- **PHP** - Langage serveur
- **MySQL** - Base de données
- **HTML/CSS** - Interface utilisateur
